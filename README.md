## 开发模式

> 全局安装 parcel

```bash
yarn global add parcel

或

yarn global add parcel-bundler
```

> 运行

```bash
yarn server
```

## 如果 parcel 报错就安装其他版本的 parcel

> 第一个不行就试试第二个，第二个不行就试试第一个，反正总有一个版本可行。

```bash
yarn global add parcel@1.9.7

或

yarn global add parcel@next
```

## 一键部署

```bash
yarn deploy
```
